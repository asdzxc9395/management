### 경비 관리 시스템

&nbsp;&nbsp; 개인 공부용 포트폴리오

<br/> 

---

#### 주요 특징​

- 회원가입, 로그인 기능
	- 메일 인증을 통해 가입 가능
- 경비 관리
	- 사용자: 경비 요청 및 내역 검색 가능(팝업)
	- 관리자: 사용자의 경비 요청 처리 및 사용자 관리
- 기타: 해당 조건에 맞는 경비 정보 엑셀 저장 가능
<br/>
